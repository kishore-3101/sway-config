# Minimal Sway Setup (🚧 WIP)

This is my personal Sway configuration built on Arch Linux.
Beginner-friendly and simple to understand.

## Features
- Minimalist Design
- Cleanly Documented
- Lightweight
- Beginner Friendly

## Keybindings
- Mod + X → Terminal
- Mod + B → Brave Browser
- Mod + F → File Manager
- Mod + D → App Launcher

## Dependencies
Install required packages:

sudo pacman -S sway alacritty rofi nautilus brave

## Installation
git clone https://github.com/kishore-3101o/sway-config.git
cd sway-config
mkdir -p ~/.config/sway
cp config ~/.config/sway/config
